Szallós Kis Csaba (1998) is a software developer with an appetite for natural language processing, machine learning and data visualisation.

He also writes poems in Hungarian.

Bio

studied computer science as BSc at the Babeș-Bolyai University between 2017 and 2020, among algorithms, data structures, grahps, linux, assembly, haskell, databases, c, c++, java, statistics, machine learning, algebra, networking, etc. the most valuable assets learned here were probably critical thinking and problem solving.

studied data analysis and modelling as MSc at the Babeș-Bolyai University between 2020 and 2022 because the subjects I enjoyed mostly during the first three years of university were natural language processing and machine learning, and here besides analyzing and visualising data the main focus was machine learning, but we also learned GPGPU programming and swarm intelligence.

working as a C++ software developer at Ringo Tech since 2019.

Projects

visual haikus - an application which writes haikus based on images. my license thesis, a multi-label image recognition written in python.

interactive map of extinct and endangered languages - a webpage which displays extinct and endangered languages all around the world. based on the Leaflet framework the goal was to create a data visualisation which can help users navigate easily while exploring languages. by clicking on their names, the appropriate wikipedia article will pop up for a more in-depth understanding.

geometric tunes - interactive new media installation written in python which was showcased at Clujotronic electro arts festival deployed on a Raspberry Pi.

interactive visualisation of the best movies based on letterboxd - using Flourish to represent the top 250 narrative feature films based on letterboxd.

rhythm - metric foot transcription applicable to Hungarian poetry initially written in java and afterwards in javascript.

Braille OCR - an application which can identify Braille-characters written by a Braille-typewriter. my masters thesis, where the model was built using python and used in Xamarin.Forms using the EmguCV dotnet framework.

next words of poets - a next word predicton project which predicts next words of beat icon, Allen Ginsberg and my favourite Hungarian poets, Pilinszky János and József Attila. Built LSTM model and experimented on both English and Hungarian corpus. GUI written in Streamlit.

multilingual text summarization web application - Django-based text summarization tool that allows users to input long articles or documents, and the application generates concise summaries. You can use extractive or abstractive summarization techniques and integrate popular NLP libraries like spaCy or GPT-based models for better results.

interactive word cloud generator - Develop a tool that generates interactive word clouds based on the frequency of words in a given text, allowing users to explore and analyze.

phylogenetic tree generator - Develop a basic tool for generating phylogenetic trees from user-provided DNA or protein sequences using simple algorithms.
